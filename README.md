## Usage
自動販売機に入力された番号に対応する写真が表示されます。

現在、見られる写真の番号は0から42までです。

数字のボタンをクリックし、Eボタンで決定してください。
Cボタンでクリアできます。

キーボード入力にも対応しています。
Enter >> Eボタン
c >> Cボタン

777を入力すると、写真をランダムに表示していくランダムモードになります。
